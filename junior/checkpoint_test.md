# Junior Checkpoint Test

Take this test around the middle of the level.

## Test Goal

Check whether you can build reliable data work, not just solve isolated exercises.

## Tasks

1. Build a CSV to PostgreSQL loader with basic validation.
2. Build one SQL report using at least 2 CTEs.
3. Use one window function correctly.
4. Add logging to one Python script.
5. Make one load idempotent.
6. Explain the grain of one dataset.

## Pass Standard

You pass if you can:

- complete the work with limited help
- explain why each check exists
- rerun your pipeline safely
- identify one weakness in your own design

## If You Fail

Spend 2 more weeks on:

- idempotent loading
- logging
- SQL CTEs
- window functions
- basic data modeling
