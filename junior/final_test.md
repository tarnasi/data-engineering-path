# Junior Final Test

Take this test at the end of the level.

## Final Project

Build one end-to-end pipeline that does all of the following:

1. Ingest data from file or API.
2. Validate the data before load.
3. Load it into PostgreSQL safely.
4. Transform it into a business-ready table or report.
5. Add at least one rerun-safe behavior.
6. Add logging.
7. Write a short design note.

## Required Explanation

You should be able to explain:

- the source to target flow
- the validation rules
- the rerun strategy
- the grain of the final output
- the main failure points
- how a stakeholder would use the result

## Pass Standard

You are ready for Mid if you can own this pipeline alone, explain the tradeoffs clearly, and improve it after one review cycle.
