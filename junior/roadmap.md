# Junior Mastery Roadmap

## Purpose

This level makes you reliable.
You stop doing one-off exercises and start building repeatable data work.

## Time

- Target: 16 to 20 weeks
- Slow path: up to 24 weeks
- Daily work: 2 to 3 hours ideal

## Scope

Focus on:

- SQL with CTEs and window functions
- Python pipeline structure
- validation before load
- idempotent scripts
- API and file ingestion
- basic modeling
- intro Airflow and intro dbt

## Task Mapping In This Repository

Use these tasks as core Junior work:

- Junior Task 6 to Junior Task 10

Use these only as preview or late Junior exposure:

- Junior Task 11 to Junior Task 20

Do not judge yourself badly if Tasks 11 to 20 feel harder.
That is expected.

## Weekly Structure

- Day 1: SQL and modeling
- Day 2: Python pipelines
- Day 3: SQL and debugging
- Day 4: Python and refactoring
- Day 5: end-to-end practice
- Day 6: review and explanation
- Day 7: rest or light review

## Mastery Targets

You should be able to:

- build a repeatable ingestion script
- validate data before loading it
- log failures clearly
- rerun a script safely
- answer business questions with stronger SQL
- explain grain, dimensions, and facts at a basic level

## Exit Condition

Do not leave this level until you can build one full mini pipeline from source to report and explain its design clearly.
