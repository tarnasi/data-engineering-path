# Beginner Final Test

Take this test at the end of the level.

## Final Project

Build a small data flow that does all of the following:

1. Read 3 source files.
2. Load them into PostgreSQL.
3. Run profiling queries for row counts, nulls, duplicates, and date ranges.
4. Answer 10 business questions in SQL.
5. Export at least 2 results to CSV.
6. Write a short report explaining what you found.

## Oral Explanation Test

You should also be able to explain:

- why you created each table
- how your script finds the files
- what problems existed in the source data
- how you know your results are correct
- what would break if the file format changes

## Pass Standard

You are ready for Junior if you can complete the project alone and explain it clearly without depending on a tutorial.
