# Beginner Checkpoint Test

Take this test around the middle of the level, usually after 5 to 6 weeks.

## Test Goal

Check whether your basics are stable enough to continue.

## Tasks

1. Load one CSV file into PostgreSQL.
2. Write 8 SQL queries using SELECT, WHERE, GROUP BY, HAVING, and JOIN.
3. Find nulls and duplicates in one table.
4. Clean one messy CSV with pandas and export the clean file.
5. Explain the meaning of primary key, foreign key, null, duplicate, and aggregation.

## Pass Standard

You pass if you can:

- complete the work without step-by-step copying
- explain your queries in your own words
- produce correct output
- rerun your script with only small edits if needed

## If You Fail

Spend 1 more week only on:

- SQL joins
- GROUP BY and aggregations
- pandas cleaning basics
- PostgreSQL loading basics
