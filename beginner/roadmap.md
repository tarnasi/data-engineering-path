# Beginner Mastery Roadmap

## Purpose

This level builds your base.
You are not trying to learn every data engineering tool.
You are trying to become stable in SQL, Python, files, and PostgreSQL.

## Time

- Target: 10 to 12 weeks
- Slow path: 14 to 16 weeks
- Daily work: 90 minutes minimum, 2.5 to 3 hours ideal

## Scope

Focus only on:

- SQL fundamentals
- Python fundamentals
- CSV and JSON handling
- pandas basics
- PostgreSQL basics
- terminal and Git basics

Ignore for now:

- Airflow
- dbt
- Spark
- Kafka
- cloud architecture
- system design

## Task Mapping In This Repository

Ignore all stretch goals.

## Weekly Structure

- Day 1: SQL
- Day 2: SQL
- Day 3: Python
- Day 4: Python
- Day 5: combined practice
- Day 6: review and mini project
- Day 7: rest or light notes only

## Mastery Targets

You should be able to:

- load a CSV into PostgreSQL
- clean a messy file with pandas
- write joins and aggregations without copying a template
- inspect nulls, duplicates, and date ranges
- explain the difference between source data and cleaned data

## Exit Condition

Do not leave this level until you can finish a small CSV to PostgreSQL project and explain every step clearly.
