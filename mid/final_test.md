# Mid Final Test

Take this test at the end of the level.

## Final Project

Build a small analytics platform that does all of the following:

1. Ingest raw data.
2. Transform it through structured layers.
3. Model it for analytics use.
4. Add dbt tests and documentation.
5. Orchestrate it with Airflow.
6. Support reruns or backfills safely.
7. Explain performance considerations.

## Required Explanation

You should be able to explain:

- why the architecture is layered this way
- how the grain was chosen
- how history is handled
- how failures are surfaced
- how quality is enforced
- how the system can scale or where it will break first

## Pass Standard

You are ready for Senior-track work if you can design, defend, and operate this system with clear tradeoffs and minimal supervision.
