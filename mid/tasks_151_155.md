## Task 151: Stream-Table Join Patterns — Enriching Events in Real Time

### Business Context
NovaMart's real-time fraud detection (Task 132) catches suspicious patterns, but it flags too many false positives because it only sees raw events. The Fraud team says: "If we could enrich each event with customer history (lifetime spend, account age, past flags) at stream time, we'd reduce false positives by 60%." This requires joining a high-velocity stream with slowly-changing reference data — a classic stream-table join.

### Objective
Implement stream-table join patterns that enrich streaming events with dimensional data, handling updates to reference data gracefully.

### Technical Requirements
- **Tools**: Flink SQL or Spark Structured Streaming, Kafka, PostgreSQL/Snowflake, Redis
- **Skills**: Stream-table joins, temporal joins, lookup caching, cache invalidation

### Step-by-Step Guidance

1. **Map join patterns required**:
   ```
   Stream-Table Join Types:
   ┌─────────────────────────────────────┐
   │  1. Lookup Join (point-in-time)     │ ← Most common
   │     Stream event + current dim row  │
   │                                     │
   │  2. Temporal Join                   │ ← For SCD accuracy
   │     Stream event + dim-as-of-time   │
   │                                     │
   │  3. Broadcast Join                  │ ← Small reference tables
   │     Stream + full table in memory   │
   └─────────────────────────────────────┘
   ```

2. **Implement a Flink SQL temporal join**:
   ```sql
   -- Create the streaming source (Kafka orders)
   CREATE TABLE order_events (
       order_id STRING,
       customer_id STRING,
       amount DECIMAL(10,2),
       event_time TIMESTAMP(3),
       WATERMARK FOR event_time AS event_time - INTERVAL '5' SECOND
   ) WITH (
       'connector' = 'kafka',
       'topic' = 'novamart.orders',
       'properties.bootstrap.servers' = 'localhost:9092',
       'format' = 'json',
       'scan.startup.mode' = 'latest-offset'
   );

   -- Create the dimension table (CDC from PostgreSQL)
   CREATE TABLE customer_dim (
       customer_id STRING,
       lifetime_spend DECIMAL(12,2),
       account_age_days INT,
       risk_score DECIMAL(3,2),
       is_vip BOOLEAN,
       update_time TIMESTAMP(3),
       WATERMARK FOR update_time AS update_time - INTERVAL '1' MINUTE,
       PRIMARY KEY (customer_id) NOT ENFORCED
   ) WITH (
       'connector' = 'kafka',
       'topic' = 'novamart.customers.cdc',
       'format' = 'debezium-json',
       'scan.startup.mode' = 'earliest-offset'
   );

   -- Temporal join: enrich each order with customer state at order time
   SELECT
       o.order_id,
       o.customer_id,
       o.amount,
       c.lifetime_spend,
       c.account_age_days,
       c.risk_score,
       c.is_vip,
       CASE
           WHEN o.amount > c.lifetime_spend * 0.5 THEN 'HIGH'
           WHEN c.account_age_days < 7 AND o.amount > 200 THEN 'MEDIUM'
           ELSE 'LOW'
       END AS fraud_risk_level
   FROM order_events o
   JOIN customer_dim FOR SYSTEM_TIME AS OF o.event_time AS c
       ON o.customer_id = c.customer_id;
   ```

3. **Implement a cached lookup join in Python** (for Spark Structured Streaming):
   ```python
   import redis
   import json
   from pyspark.sql import SparkSession
   from pyspark.sql.functions import udf, col, from_json, struct
   from pyspark.sql.types import (
       StructType, StructField, StringType,
       DecimalType, IntegerType, BooleanType
   )

   redis_client = redis.Redis(host='localhost', port=6379, db=0)

   def get_customer_dim(customer_id: str) -> dict:
       """Lookup customer dimension from Redis cache."""
       cached = redis_client.get(f"customer:{customer_id}")
       if cached:
           return json.loads(cached)
       return {
           "lifetime_spend": 0, "account_age_days": 0,
           "risk_score": 0.5, "is_vip": False
       }

   # Register as UDF
   customer_schema = StructType([
       StructField("lifetime_spend", DecimalType(12, 2)),
       StructField("account_age_days", IntegerType()),
       StructField("risk_score", DecimalType(3, 2)),
       StructField("is_vip", BooleanType()),
   ])

   lookup_udf = udf(get_customer_dim, customer_schema)

   # Read stream
   spark = SparkSession.builder.appName("EnrichedOrders").getOrCreate()
   orders = (spark.readStream
       .format("kafka")
       .option("subscribe", "novamart.orders")
       .load()
   )

   # Enrich with dimension lookup
   enriched = (orders
       .withColumn("customer", lookup_udf(col("customer_id")))
       .select("order_id", "customer_id", "amount",
               col("customer.lifetime_spend"),
               col("customer.account_age_days"),
               col("customer.risk_score"))
   )
   ```

4. **Build a cache refresh pipeline** to keep Redis in sync:
   ```python
   # cache_refresher.py — Runs as a Kafka consumer
   from kafka import KafkaConsumer
   import redis
   import json

   consumer = KafkaConsumer(
       'novamart.customers.cdc',
       bootstrap_servers='localhost:9092',
       value_deserializer=lambda m: json.loads(m.decode('utf-8')),
       group_id='cache-refresh',
       auto_offset_reset='earliest'
   )

   r = redis.Redis(host='localhost', port=6379, db=0)

   for message in consumer:
       payload = message.value
       if payload.get('op') in ('c', 'u', 'r'):  # create, update, read
           after = payload['after']
           customer_id = after['customer_id']
           r.setex(
               f"customer:{customer_id}",
               3600,  # TTL: 1 hour
               json.dumps({
                   "lifetime_spend": float(after['lifetime_spend']),
                   "account_age_days": after['account_age_days'],
                   "risk_score": float(after['risk_score']),
                   "is_vip": after['is_vip']
               })
           )
       elif payload.get('op') == 'd':  # delete
           r.delete(f"customer:{payload['before']['customer_id']}")
   ```

### Common Mistakes

## Task 152: Statistical Process Control for Data Pipelines

### Business Context
NovaMart's data quality checks (Tasks 112, 137, 140) catch obvious issues — nulls, schema violations, constraint failures. But they miss *subtle drift*. Last week, the average order value gradually dropped 15% over 3 days. No single batch failed any check — each was within historical bounds individually. But the *trend* was a bug in the pricing service. Priya says: "We need statistical process control — the same way factories monitor production lines."

### Objective
Implement Statistical Process Control (SPC) for data pipelines that detects gradual drift, sudden shifts, and unusual patterns using control charts and statistical rules.

### Technical Requirements
- **Tools**: Python (scipy, numpy, pandas), Snowflake, Airflow, Grafana
- **Skills**: Control charts, Western Electric rules, CUSUM, trend detection

### Step-by-Step Guidance

1. **Understand SPC fundamentals for data**:
   ```
   Control Chart Anatomy:
   ┌──────────────────────────────────────────┐
   │  UCL ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ Upper    │ +3σ
   │                                          │
   │  UWL ─ · ─ · ─ · ─ · ─ · ─ · Warning   │ +2σ
   │          *    *                          │
   │  CL  ═══*══*═══*══*══*═══════ Center    │ Mean
   │              *       *  *                │
   │  LWL ─ · ─ · ─ · ─ · ─ · ─ · Warning   │ -2σ
   │                          *               │
   │  LCL ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ Lower    │ -3σ
   └──────────────────────────────────────────┘
   ```

2. **Build the SPC engine**:
   ```python
   # spc_engine.py
   import numpy as np
   import pandas as pd
   from dataclasses import dataclass, field
   from enum import Enum
   from typing import Optional

   class ViolationType(Enum):
       BEYOND_3_SIGMA = "Point beyond 3σ"
       TWO_OF_THREE_BEYOND_2_SIGMA = "2 of 3 consecutive points beyond 2σ"
       FOUR_OF_FIVE_BEYOND_1_SIGMA = "4 of 5 consecutive beyond 1σ"
       EIGHT_CONSECUTIVE_SAME_SIDE = "8 consecutive points on same side of center"
       SIX_TRENDING = "6 consecutive increasing or decreasing"
       FIFTEEN_WITHIN_1_SIGMA = "15 consecutive within 1σ (reduced variation)"

   @dataclass
   class ControlLimits:
       center: float
       ucl: float          # Upper Control Limit (+3σ)
       lcl: float          # Lower Control Limit (-3σ)
       uwl: float          # Upper Warning Limit (+2σ)
       lwl: float          # Lower Warning Limit (-2σ)
       one_sigma_upper: float
       one_sigma_lower: float

   @dataclass
   class Violation:
       rule: ViolationType
       index: int
       value: float
       severity: str       # "warning" or "alarm"

   class SPCMonitor:
       def __init__(self, baseline_data: pd.Series, metric_name: str):
           self.metric_name = metric_name
           self.mean = baseline_data.mean()
           self.std = baseline_data.std()
           self.limits = ControlLimits(
               center=self.mean,
               ucl=self.mean + 3 * self.std,
               lcl=self.mean - 3 * self.std,
               uwl=self.mean + 2 * self.std,
               lwl=self.mean - 2 * self.std,
               one_sigma_upper=self.mean + self.std,
               one_sigma_lower=self.mean - self.std,
           )

       def check_western_electric_rules(
           self, data: pd.Series
       ) -> list[Violation]:
           violations = []
           values = data.values

           for i, val in enumerate(values):
               # Rule 1: Any point beyond 3σ
               if val > self.limits.ucl or val < self.limits.lcl:
                   violations.append(Violation(
                       ViolationType.BEYOND_3_SIGMA, i, val, "alarm"
                   ))

           # Rule 2: 2 of 3 consecutive beyond 2σ (same side)
           for i in range(2, len(values)):
               window = values[i-2:i+1]
               above = sum(1 for v in window if v > self.limits.uwl)
               below = sum(1 for v in window if v < self.limits.lwl)
               if above >= 2 or below >= 2:
                   violations.append(Violation(
                       ViolationType.TWO_OF_THREE_BEYOND_2_SIGMA,
                       i, values[i], "alarm"
                   ))

           # Rule 5: 6 consecutive trending (increasing or decreasing)
           for i in range(5, len(values)):
               window = values[i-5:i+1]
               diffs = np.diff(window)
               if all(d > 0 for d in diffs) or all(d < 0 for d in diffs):
                   violations.append(Violation(
                       ViolationType.SIX_TRENDING,
                       i, values[i], "warning"
                   ))

           return violations

       def check_cusum(
           self, data: pd.Series, threshold: float = 4.0,
           drift: float = 0.5
       ) -> list[Violation]:
           """CUSUM detects small persistent shifts."""
           violations = []
           s_pos, s_neg = 0.0, 0.0
           k = drift * self.std

           for i, val in enumerate(data):
               z = (val - self.mean) / self.std
               s_pos = max(0, s_pos + z - k)
               s_neg = max(0, s_neg - z - k)

               if s_pos > threshold or s_neg > threshold:
                   violations.append(Violation(
                       ViolationType.SIX_TRENDING,
                       i, val, "alarm"
                   ))
                   s_pos, s_neg = 0.0, 0.0  # Reset after detection

           return violations
   ```

3. **Integrate with Airflow as a daily quality check**:
   ```python
   # dags/spc_quality_check.py
   from airflow.decorators import dag, task
   from datetime import datetime
   import pandas as pd

   @dag(schedule="@daily", start_date=datetime(2025, 1, 1), catchup=False)
   def spc_daily_check():

       @task
       def fetch_metrics(ds=None) -> dict:
           """Fetch today's pipeline metrics from Snowflake."""
           from snowflake.connector import connect
           conn = connect(account="novamart", user="airflow", ...)
           metrics = {}
           queries = {
               "order_count": "SELECT COUNT(*) FROM fact_orders WHERE order_date = %s",
               "avg_order_value": "SELECT AVG(total_amount) FROM fact_orders WHERE order_date = %s",
               "null_rate_email": "SELECT AVG(CASE WHEN email IS NULL THEN 1.0 ELSE 0.0 END) FROM dim_customers",
               "load_latency_sec": "SELECT MAX(loaded_at) - MAX(event_time) FROM fact_orders WHERE order_date = %s",
           }
           for name, query in queries.items():
               cur = conn.cursor()
               cur.execute(query, (ds,))
               metrics[name] = cur.fetchone()[0]
           conn.close()
           return metrics

       @task
       def run_spc(metrics: dict, ds=None):
           """Run SPC checks against 30-day baseline."""
           from spc_engine import SPCMonitor
           # Load 30-day baseline from Snowflake
           baseline = load_baseline(metric_name, days=30)
           all_violations = []

           for metric_name, current_value in metrics.items():
               monitor = SPCMonitor(baseline[metric_name], metric_name)
               today_series = pd.Series([current_value])
               full = pd.concat([baseline[metric_name], today_series])
               violations = monitor.check_western_electric_rules(full)
               violations += monitor.check_cusum(full)
               all_violations.extend(
                   [(metric_name, v) for v in violations
                    if v.index == len(full) - 1]  # Only today's violations
               )

           if all_violations:
               send_spc_alert(all_violations)
           return {"violations": len(all_violations)}

       metrics = fetch_metrics()
       run_spc(metrics)

   spc_daily_check()
   ```

3. **Build the pipeline generator**:
   ```python
   # platform/generator.py
   import yaml
   from pathlib import Path
   from jinja2 import Environment, FileSystemLoader

   class PipelineGenerator:
       def __init__(self, template_dir: str = "platform/templates"):
           self.env = Environment(
               loader=FileSystemLoader(template_dir),
               keep_trailing_newline=True
           )

       def generate(self, request_path: str, output_dir: str):
           with open(request_path) as f:
               config = yaml.safe_load(f)

           pipeline_name = config['metadata']['name']
           out = Path(output_dir) / pipeline_name
           out.mkdir(parents=True, exist_ok=True)

           # 1. Generate Airflow DAG
           dag_template = self.env.get_template("airflow_dag.py.j2")
           (out / "dags" / f"{pipeline_name}.py").parent.mkdir(
               parents=True, exist_ok=True
           )
           (out / "dags" / f"{pipeline_name}.py").write_text(
               dag_template.render(config=config)
           )

           # 2. Generate dbt models
           for model in config['spec']['transform']['staging']:
               stg_template = self.env.get_template("dbt_staging.sql.j2")
               (out / "models" / "staging").mkdir(parents=True, exist_ok=True)
               (out / "models" / "staging" / f"{model}.sql").write_text(
                   stg_template.render(
                       model_name=model, config=config
                   )
               )

           # 3. Generate dbt schema YAML (tests)
           schema_template = self.env.get_template("dbt_schema.yml.j2")
           (out / "models" / "schema.yml").write_text(
               schema_template.render(config=config)
           )

           # 4. Generate Terraform for infrastructure
           tf_template = self.env.get_template("terraform.tf.j2")
           (out / "infra" / "main.tf").parent.mkdir(parents=True, exist_ok=True)
           (out / "infra" / "main.tf").write_text(
               tf_template.render(config=config)
           )

           # 5. Generate Grafana dashboard
           dash_template = self.env.get_template("grafana_dashboard.json.j2")
           (out / "monitoring" / "dashboard.json").parent.mkdir(
               parents=True, exist_ok=True
           )
           (out / "monitoring" / "dashboard.json").write_text(
               dash_template.render(config=config)
           )

           return out
   ```

   ```python
   import httpx

       title="NovaMart Data Platform API",
       version="2.0.0",

   )

   # Service registry
   SERVICES = {
       "metrics": {"url": "http://metrics-api:8001", "prefix": "/v2/metrics"},
       "features": {"url": "http://features-api:8002", "prefix": "/v2/features"},
       "pipelines": {"url": "http://pipeline-api:8003", "prefix": "/v2/pipelines"},
       "catalog": {"url": "http://catalog-api:8004", "prefix": "/v2/catalog"},
   }

   rate_limiter = RateLimiter(
       redis_client=redis.Redis(host='localhost', port=6379),
       default_limit=100,   # requests per minute
       default_window=60,
   )

   # Tier-based rate limits
   RATE_TIERS = {
       "free": {"rpm": 60, "daily": 1000},
       "team": {"rpm": 300, "daily": 50000},
       "platform": {"rpm": 1000, "daily": 500000},
   }

   @app.middleware("http")
   async def gateway_middleware(request: Request, call_next):
       start = time.time()
       response = await call_next(request)
       duration = time.time() - start
       response.headers["X-Response-Time"] = f"{duration:.3f}s"
       response.headers["X-Gateway-Version"] = "2.0.0"
       return response

   @app.get("/v2/metrics/{path:path}")
   async def proxy_metrics(
       path: str, request: Request,
       user=Depends(get_current_user)
   ):
       await rate_limiter.check(user.team, RATE_TIERS[user.tier])
       async with httpx.AsyncClient() as client:
           resp = await client.get(
               f"{SERVICES['metrics']['url']}/{path}",
               params=request.query_params,
               headers={"X-User": user.email, "X-Team": user.team}
           )
       return resp.json()
   ```

3. **Implement sliding window rate limiting**:
   ```python
   # gateway/rate_limiter.py
   import redis
   import time
   from fastapi import HTTPException

   class RateLimiter:
       def __init__(self, redis_client: redis.Redis,
                    default_limit: int = 100, default_window: int = 60):
           self.redis = redis_client
           self.default_limit = default_limit
           self.default_window = default_window

       async def check(self, team: str, tier_config: dict):
           key = f"ratelimit:{team}:{int(time.time()) // 60}"
           current = self.redis.incr(key)
           if current == 1:
               self.redis.expire(key, self.default_window + 10)

           if current > tier_config["rpm"]:
               raise HTTPException(
                   status_code=429,
                   detail={
                       "error": "Rate limit exceeded",
                       "limit": tier_config["rpm"],
                       "retry_after": 60 - (int(time.time()) % 60)
                   }
               )
   ```

2. **Implement the SDK core**:
   ```python
   # novamart_data/client.py
   import httpx
   from typing import Optional
   from novamart_data.config import Config
   from novamart_data.metrics import MetricsClient
   from novamart_data.features import FeaturesClient
   from novamart_data.pipelines import PipelinesClient
   from novamart_data.catalog import CatalogClient

   class DataPlatformClient:
       def __init__(
           self,
           api_key: str,
           environment: str = "production",
           base_url: Optional[str] = None,
           timeout: float = 30.0,
           max_retries: int = 3,
       ):
           self.config = Config(
               api_key=api_key,
               environment=environment,
               base_url=base_url or self._default_url(environment),
               timeout=timeout,
               max_retries=max_retries,
           )
           self._http = self._build_client()
           self.metrics = MetricsClient(self._http, self.config)
           self.features = FeaturesClient(self._http, self.config)
           self.pipelines = PipelinesClient(self._http, self.config)
           self.catalog = CatalogClient(self._http, self.config)

       def _default_url(self, env: str) -> str:
           urls = {
               "production": "https://data-api.novamart.internal",
               "staging": "https://data-api-staging.novamart.internal",
           }
           return urls[env]

       def _build_client(self) -> httpx.Client:
           transport = httpx.HTTPTransport(retries=self.config.max_retries)
           return httpx.Client(
               base_url=self.config.base_url,
               headers={
                   "Authorization": f"Bearer {self.config.api_key}",
                   "User-Agent": f"novamart-sdk/1.0.0",
                   "Content-Type": "application/json",
               },
               timeout=self.config.timeout,
               transport=transport,
           )

       def health(self) -> dict:
           resp = self._http.get("/v2/health")
           resp.raise_for_status()
           return resp.json()
   ```

3. **Implement the Metrics sub-client with typed responses**:
   ```python
   # novamart_data/metrics.py
   import httpx
   import pandas as pd
   from pydantic import BaseModel
   from typing import Optional
   from datetime import date

   class MetricResult(BaseModel):
       metric_name: str
       time_grain: str
       dimensions: list[str]
       data: list[dict]

       def to_dataframe(self) -> pd.DataFrame:
           return pd.DataFrame(self.data)

       def to_dict(self) -> list[dict]:
           return self.data

   class MetricsClient:
       def __init__(self, http: httpx.Client, config):
           self._http = http
           self._config = config

       def query(
           self,
           metric: str,
           dimensions: Optional[list[str]] = None,
           time_grain: str = "daily",
           start_date: Optional[str] = None,
           end_date: Optional[str] = None,
           filters: Optional[dict] = None,
       ) -> MetricResult:
           params = {
               "metric": metric,
               "grain": time_grain,
               "dimensions": ",".join(dimensions or []),
           }
           if start_date:
               params["start_date"] = start_date
           if end_date:
               params["end_date"] = end_date
           if filters:
               for k, v in filters.items():
                   params[f"filter_{k}"] = v

           resp = self._http.get("/v2/metrics/query", params=params)
           resp.raise_for_status()
           return MetricResult(**resp.json())

       def list_metrics(self, domain: Optional[str] = None) -> list[dict]:
           params = {"domain": domain} if domain else {}
           resp = self._http.get("/v2/metrics/list", params=params)
           resp.raise_for_status()
           return resp.json()["metrics"]
   ```

4. **Add comprehensive error handling**:

   ```python
   # novamart_data/exceptions.py
   class DataPlatformError(Exception):
       """Base exception for SDK errors."""
       def __init__(self, message: str, status_code: int = None, response=None):
           self.status_code = status_code
           self.response = response
           super().__init__(message)

   class AuthenticationError(DataPlatformError):
       """Invalid or expired API key."""

   class RateLimitError(DataPlatformError):
       """Rate limit exceeded."""
       def __init__(self, retry_after: int, **kwargs):
           self.retry_after = retry_after
           super().__init__(
               f"Rate limit exceeded. Retry after {retry_after}s", **kwargs
           )

   class NotFoundError(DataPlatformError):
       """Requested resource not found."""

   class ValidationError(DataPlatformError):
       """Invalid request parameters."""
   ```
