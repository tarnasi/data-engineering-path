## Task 236: Feature Store — Bridging Data Engineering and Machine Learning

### Business Context
NovaMart's ML team (4 engineers) spends 75% of their time on feature engineering: extracting order history, computing rolling averages, building customer profiles. They re-derive the same features for every model. Worse, the features they compute in Python notebooks drift from the production SQL equivalents, causing training-serving skew. The ML lead complains: "I need a feature store. My team shouldn't be writing SQL — your platform should provide features as a service."

### Objective
Build a feature store that serves precomputed features to ML models, ensuring consistency between training and inference (no train-serve skew).

### Technical Requirements
- **Tools**: Python, SQL, Feast (feature store framework), Snowflake, Redis
- **Skills**: Feature engineering, ML systems, online/offline serving, point-in-time correctness

### Step-by-Step Guidance

1. **Audit current feature duplication**:

   ```markdown
   ## Feature Duplication Audit

   | Feature | ML Notebook (Python) | dbt Model (SQL) | Drift? |
   |---------|---------------------|------------------|--------|
   | customer_lifetime_value | pandas rolling SUM, 365 days | SUM() OVER last 365 days | ✅ Off by ~2% (timezone handling) |
   | avg_order_value_30d | Python, includes cancelled | SQL, excludes cancelled | ❌ Systematic 5% overcount in ML |
   | days_since_last_order | datetime.now() - last_order | DATEDIFF(day, last_order, CURRENT_DATE()) | ✅ Training uses stale snapshots |
   | category_preference_vector | TF-IDF on categories | Not in SQL | N/A |
   | is_high_value_customer | Revenue > $500 | Revenue > $1000 | ❌ Different thresholds! |

   **Impact**: Fraud model has 12% higher false positive rate in prod than in training.
   Root cause: 3 features with train-serve skew.
   ```

3. **Build the feature computation pipeline**:
   ```sql
   -- feature_store/sql/customer_features.sql
   -- This dbt model is the SINGLE SOURCE for customer features.
   -- Both training and inference read from here.

   WITH order_history AS (
       SELECT
           customer_id,
           COUNT(*) AS lifetime_order_count,
           SUM(total_amount_usd) AS lifetime_revenue_usd,
           AVG(CASE WHEN order_date >= DATEADD(day, -30, CURRENT_DATE())
                    THEN total_amount_usd END) AS avg_order_value_30d,
           DATEDIFF(day, MAX(order_date), CURRENT_DATE()) AS days_since_last_order,
           MAX(order_date) AS last_order_date
       FROM {{ ref('fact_orders') }}
       WHERE status IN ('shipped', 'delivered')
       GROUP BY customer_id
   ),

   category_prefs AS (
       SELECT
           customer_id,
           product_category AS preferred_category,
           ROW_NUMBER() OVER (
               PARTITION BY customer_id
               ORDER BY COUNT(*) DESC
           ) AS rn
       FROM {{ ref('fact_order_items') }}
       GROUP BY customer_id, product_category
   ),

   return_stats AS (
       SELECT
           customer_id,
           COUNT(CASE WHEN is_returned THEN 1 END)::FLOAT
               / NULLIF(COUNT(*), 0) AS return_rate,
           AVG(delivery_days) AS avg_delivery_days
       FROM {{ ref('fact_orders') }}
       GROUP BY customer_id
   )

   SELECT
       oh.customer_id,
       oh.lifetime_order_count,
       oh.lifetime_revenue_usd,
       oh.avg_order_value_30d,
       oh.days_since_last_order,
       cp.preferred_category,
       c.preferred_channel,
       rs.return_rate,
       rs.avg_delivery_days,
       CASE WHEN oh.lifetime_revenue_usd > 500 THEN 1 ELSE 0 END AS is_high_value,
       c.region,
       DATEDIFF(day, c.created_at, CURRENT_DATE()) AS account_age_days,
       CURRENT_TIMESTAMP() AS feature_timestamp
   FROM order_history oh
   LEFT JOIN category_prefs cp ON oh.customer_id = cp.customer_id AND cp.rn = 1
   LEFT JOIN return_stats rs ON oh.customer_id = rs.customer_id
   LEFT JOIN {{ ref('dim_customers') }} c ON oh.customer_id = c.customer_id
   ```

4. **Implement online serving with Redis**:
   ```python
   # feature_store/online_serving.py
   import redis
   import json
   from typing import Optional

   class OnlineFeatureStore:
       """Serve features with <10ms latency for real-time inference."""

       def __init__(self, redis_host: str = "localhost", redis_port: int = 6379):
           self.redis = redis.Redis(host=redis_host, port=redis_port,
                                     decode_responses=True)
           self.FEATURE_PREFIX = "features:customer:"
           self.TTL_SECONDS = 86400  # 24 hours

       def materialize_features(self, features_batch: list[dict]):
           """Push batch features from Snowflake to Redis."""
           pipe = self.redis.pipeline()
           for row in features_batch:
               key = f"{self.FEATURE_PREFIX}{row['customer_id']}"
               pipe.setex(key, self.TTL_SECONDS, json.dumps(row))
           pipe.execute()

       def get_features(self, customer_id: str,
                        feature_names: Optional[list[str]] = None) -> dict:
           """Get features for a customer (<10ms)."""
           key = f"{self.FEATURE_PREFIX}{customer_id}"
           raw = self.redis.get(key)
           if not raw:
               return {"customer_id": customer_id, "features": None,
                       "source": "miss"}

           features = json.loads(raw)
           if feature_names:
               features = {k: v for k, v in features.items()
                          if k in feature_names or k == "customer_id"}
           return {"customer_id": customer_id, "features": features,
                   "source": "redis"}

       def get_training_data(self, customer_ids: list[str],
                             point_in_time: str) -> list[dict]:
           """Get historical features for training (point-in-time correct)."""
           # For training, query Snowflake with AS OF timestamp
           # This prevents data leakage (using future features for past labels)
           query = f"""
               SELECT *
               FROM GOLD.CUSTOMER_FEATURES
               FOR SYSTEM_TIME AS OF '{point_in_time}'::TIMESTAMP
               WHERE customer_id IN ({','.join(f"'{c}'" for c in customer_ids)})
           """
           # Execute against Snowflake and return
           return self._execute_snowflake(query)
   ```

   ```python
   # privacy/pii_scanner.py
   import re
   from dataclasses import dataclass
   from enum import Enum

   class PIICategory(Enum):
       DIRECT_IDENTIFIER = "direct"      # name, email, phone
       QUASI_IDENTIFIER = "quasi"         # zip, birthdate, gender
       SENSITIVE = "sensitive"            # health, financial, racial
       DERIVED = "derived"               # ML predictions about a person

   @dataclass
   class PIIColumn:
       database: str
       schema: str
       table: str
       column: str
       pii_category: PIICategory
       detection_method: str  # "pattern", "name", "manual"
       confidence: float
       regulation: list[str]  # ["GDPR", "CCPA", "LGPD"]

   class PIIScanner:
       NAME_PATTERNS = {
           "email": (r"email|e_mail|email_address", PIICategory.DIRECT_IDENTIFIER),
           "phone": (r"phone|mobile|tel|cell", PIICategory.DIRECT_IDENTIFIER),
           "name": (r"first_name|last_name|full_name|customer_name",
                    PIICategory.DIRECT_IDENTIFIER),
           "address": (r"street|address|city|state|zip|postal",
                       PIICategory.QUASI_IDENTIFIER),
           "ip": (r"ip_address|ip_addr|client_ip", PIICategory.QUASI_IDENTIFIER),
           "ssn": (r"ssn|social_security|tax_id", PIICategory.SENSITIVE),
           "dob": (r"birth|dob|date_of_birth", PIICategory.QUASI_IDENTIFIER),
           "cc": (r"credit_card|card_number|pan", PIICategory.SENSITIVE),
       }

       def scan_schema(self, snowflake_conn, database: str,
                        schema: str) -> list[PIIColumn]:
           """Scan all columns in a schema for PII."""
           columns = snowflake_conn.cursor().execute(f"""
               SELECT table_name, column_name, data_type
               FROM {database}.information_schema.columns
               WHERE table_schema = '{schema}'
           """).fetchall()

           pii_found = []
           for table, column, dtype in columns:
               for pii_type, (pattern, category) in self.NAME_PATTERNS.items():
                   if re.search(pattern, column, re.IGNORECASE):
                       pii_found.append(PIIColumn(
                           database=database,
                           schema=schema,
                           table=table,
                           column=column,
                           pii_category=category,
                           detection_method="name_pattern",
                           confidence=0.9,
                           regulation=self._applicable_regulations(category),
                       ))
                       break
           return pii_found

       def _applicable_regulations(self, category: PIICategory) -> list[str]:
           if category == PIICategory.DIRECT_IDENTIFIER:
               return ["GDPR", "CCPA", "LGPD"]
           elif category == PIICategory.SENSITIVE:
               return ["GDPR", "CCPA", "LGPD", "HIPAA"]
           return ["GDPR"]
   ```

2. **Build the GDPR deletion pipeline**:
   ```python
   # privacy/gdpr_deletion.py
   from dataclasses import dataclass, field
   from datetime import datetime

   @dataclass
   class DeletionRequest:
       request_id: str
       customer_id: str
       customer_email: str
       regulation: str       # "GDPR", "CCPA"
       requested_at: datetime
       deadline: datetime     # 30 days from request
       status: str = "pending"
       deletion_log: list = field(default_factory=list)

   class GDPRDeletionPipeline:
       """Automated right-to-be-forgotten pipeline."""

       def __init__(self, snowflake_conn, pii_map: list[PIIColumn],
                    kafka_admin, s3_client):
           self.conn = snowflake_conn
           self.pii_map = pii_map
           self.kafka = kafka_admin
           self.s3 = s3_client

       def execute_deletion(self, request: DeletionRequest) -> dict:
           """Delete all customer data across every system."""
           results = {
               "request_id": request.request_id,
               "customer_id": request.customer_id,
               "systems_processed": [],
               "errors": [],
           }

           # Step 1: Delete from Snowflake (all tables with PII)
           sf_result = self._delete_from_snowflake(request.customer_id)
           results["systems_processed"].append(sf_result)

           # Step 2: Publish tombstone records to Kafka
           kafka_result = self._publish_kafka_tombstones(request.customer_id)
           results["systems_processed"].append(kafka_result)

           # Step 3: Delete from S3 raw data
           s3_result = self._delete_from_s3(request.customer_id)
           results["systems_processed"].append(s3_result)

           # Step 4: Log the deletion (required for audit)
           self._log_deletion(request, results)

           return results

       def _delete_from_snowflake(self, customer_id: str) -> dict:
           """Delete customer data from all Snowflake tables."""
           tables_affected = set()
           for pii_col in self.pii_map:
               if pii_col.pii_category == PIICategory.DIRECT_IDENTIFIER:
                   # For fact tables: anonymize (replace with hash)
                   if "fact_" in pii_col.table.lower():
                       self.conn.cursor().execute(f"""
                           UPDATE {pii_col.database}.{pii_col.schema}.{pii_col.table}
                           SET {pii_col.column} = SHA2('{customer_id}' || 'salt')
                           WHERE customer_id = %s
                       """, (customer_id,))
                   else:
                       # For dimension tables: hard delete
                       self.conn.cursor().execute(f"""
                           DELETE FROM {pii_col.database}.{pii_col.schema}.{pii_col.table}
                           WHERE customer_id = %s
                       """, (customer_id,))
                   tables_affected.add(
                       f"{pii_col.database}.{pii_col.schema}.{pii_col.table}"
                   )

           return {
               "system": "snowflake",
               "tables_affected": len(tables_affected),
               "status": "completed",
           }

       def _publish_kafka_tombstones(self, customer_id: str) -> dict:
           """Publish tombstone records to compact topics."""
           topics = ["customer-events", "order-events", "activity-events"]
           for topic in topics:
               # Tombstone: same key, null value → Kafka compaction deletes it
               self.kafka.produce(topic, key=customer_id, value=None)
           return {
               "system": "kafka",
               "topics_processed": len(topics),
               "status": "tombstones_published",
           }

       def _delete_from_s3(self, customer_id: str) -> dict:
           """Delete raw data files containing customer data."""
           prefixes = [
               f"raw/customers/{customer_id}/",
               f"raw/orders/customer={customer_id}/",
           ]
           deleted = 0
           for prefix in prefixes:
               objects = self.s3.list_objects_v2(
                   Bucket="novamart-data-lake", Prefix=prefix
               )
               for obj in objects.get("Contents", []):
                   self.s3.delete_object(
                       Bucket="novamart-data-lake", Key=obj["Key"]
                   )
                   deleted += 1
           return {
               "system": "s3",
               "files_deleted": deleted,
               "status": "completed",
           }

       def _log_deletion(self, request: DeletionRequest, results: dict):
           """Audit log: legally required to prove deletion happened."""
           self.conn.cursor().execute("""
               INSERT INTO COMPLIANCE.DELETION_LOG
               (request_id, customer_id, regulation, requested_at,
                completed_at, systems_processed, status)
               VALUES (%s, %s, %s, %s, CURRENT_TIMESTAMP(), %s, %s)
           """, (
               request.request_id,
               request.customer_id,
               request.regulation,
               request.requested_at,
               str(results["systems_processed"]),
               "completed" if not results["errors"] else "partial",
           ))
   ```

   ```sql
   -- EU data stays in EU region
   CREATE OR REPLACE ROW ACCESS POLICY eu_data_residency
   AS (region VARCHAR) RETURNS BOOLEAN ->
       CASE
           WHEN CURRENT_ROLE() IN ('EU_DATA_ADMIN', 'GDPR_ADMIN') THEN TRUE
           WHEN region = 'EU' AND CURRENT_ROLE() NOT LIKE '%EU%' THEN FALSE
           ELSE TRUE
       END;

   -- Apply to fact tables
   ALTER TABLE GOLD.FACT_ORDERS
       ADD ROW ACCESS POLICY eu_data_residency ON (region);

   -- Dynamic masking for PII fields
   CREATE OR REPLACE MASKING POLICY pii_email_mask
   AS (val STRING) RETURNS STRING ->
       CASE
           WHEN CURRENT_ROLE() IN ('PII_ADMIN', 'COMPLIANCE') THEN val
           WHEN CURRENT_ROLE() LIKE '%ANALYST%' THEN
               REGEXP_REPLACE(val, '(.{2}).*(@.*)', '\\1***\\2')
   ```

2. **Build the chaos framework**:
   ```python
   # chaos/framework.py
   from dataclasses import dataclass, field
   from datetime import datetime
   from enum import Enum
   import time
   import logging

   logger = logging.getLogger("chaos")

   class ExperimentStatus(Enum):
       PENDING = "pending"
       RUNNING = "running"
       PASSED = "passed"
       FAILED = "failed"
       ABORTED = "aborted"

   @dataclass
   class ChaosExperiment:
       name: str
       description: str
       failure_type: str
       target_system: str
       inject_action: str       # What we break
       expected_behavior: str   # What should happen
       abort_condition: str     # When to stop immediately
       max_duration_seconds: int
       status: ExperimentStatus = ExperimentStatus.PENDING
       results: dict = field(default_factory=dict)
       started_at: datetime = None
       ended_at: datetime = None

   class ChaosRunner:
       def __init__(self):
           self.experiments = []
           self.rollback_actions = []

       def register_experiment(self, experiment: ChaosExperiment):
           self.experiments.append(experiment)

       def run_experiment(self, experiment: ChaosExperiment,
                          inject_fn, verify_fn, rollback_fn) -> dict:
           """
           Run a chaos experiment with safety guardrails.
           1. Inject failure
           2. Observe system behavior
           3. Verify expectations
           4. Rollback (always, even on failure)
           """
           experiment.started_at = datetime.utcnow()
           experiment.status = ExperimentStatus.RUNNING
           logger.info(f"🔥 Starting chaos experiment: {experiment.name}")

           try:
               # Safety check: only run in staging
               if not self._verify_environment():
                   raise RuntimeError("Chaos experiments only run in staging!")

               # Step 1: Take baseline measurements
               baseline = self._capture_baseline(experiment.target_system)

               # Step 2: Inject failure
               logger.info(f"Injecting: {experiment.inject_action}")
               inject_fn()

               # Step 3: Wait and observe
               duration = 0
               while duration < experiment.max_duration_seconds:
                   if self._should_abort(experiment):
                       experiment.status = ExperimentStatus.ABORTED
                       logger.warning("ABORT: Safety condition triggered")
                       break
                   time.sleep(5)
                   duration += 5

               # Step 4: Verify expected behavior
               verification = verify_fn(baseline)
               experiment.results = verification

               if verification.get("passed"):
                   experiment.status = ExperimentStatus.PASSED
                   logger.info(f"✅ Experiment PASSED: {experiment.name}")
               else:
                   experiment.status = ExperimentStatus.FAILED
                   logger.warning(
                       f"❌ Experiment FAILED: {experiment.name} — "
                       f"{verification.get('reason')}"
                   )

           except Exception as e:
               experiment.status = ExperimentStatus.FAILED
               experiment.results = {"error": str(e)}
               logger.error(f"Experiment error: {e}")

           finally:
               # ALWAYS rollback
               logger.info(f"Rolling back: {experiment.name}")
               rollback_fn()
               experiment.ended_at = datetime.utcnow()

           return {
               "experiment": experiment.name,
               "status": experiment.status.value,
               "duration_seconds": (
                   experiment.ended_at - experiment.started_at
               ).total_seconds(),
               "results": experiment.results,
           }

       def _verify_environment(self) -> bool:
           """Safety: only chaos in staging."""
           import os
           return os.getenv("ENVIRONMENT", "").lower() == "staging"

       def _capture_baseline(self, target: str) -> dict:
           """Capture current metrics before injection."""
           return {
               "timestamp": datetime.utcnow().isoformat(),
               "target": target,
               "consumer_lag": self._get_consumer_lag(),
               "pipeline_status": "healthy",
           }

       def _should_abort(self, experiment: ChaosExperiment) -> bool:
           """Check if abort conditions are met."""
           # Abort if data loss detected in production
           return False  # Implement real checks

       def _get_consumer_lag(self) -> dict:
           return {}  # Implement Kafka consumer lag check
   ```

   ```python
   from chaos.framework import ChaosExperiment, ChaosRunner

   def kafka_broker_failure_experiment(chaos_runner: ChaosRunner):
       experiment = ChaosExperiment(
           name="kafka_broker_failure",
           description="Kill one Kafka broker; verify consumers rebalance",
           failure_type="F-01",
           target_system="kafka",
           inject_action="Stop Kafka broker-2 container",
           expected_behavior=(
   ```

   ```python
   class CatalogCrawler:
       """Automatically discover and catalog all data assets."""

       def __init__(self, snowflake_conn, kafka_admin, dbt_manifest_path: str):
           self.sf = snowflake_conn
           self.kafka = kafka_admin
           self.dbt_manifest = dbt_manifest_path

       def crawl_snowflake(self) -> list[DatasetMetadata]:
           """Discover all tables and views in Snowflake."""
           tables = self.sf.cursor().execute("""
               SELECT
                   t.table_catalog, t.table_schema, t.table_name,
                   t.table_type, t.row_count, t.bytes,
                   t.last_altered, t.comment
               FROM information_schema.tables t
               WHERE t.table_schema NOT IN ('INFORMATION_SCHEMA')
               ORDER BY t.table_schema, t.table_name
           """).fetchall()

           datasets = []
           for row in tables:
               catalog, schema, name, ttype, rows, size, updated, comment = row

               # Get column details
               columns = self.sf.cursor().execute(f"""
                   SELECT column_name, data_type, comment
                   FROM information_schema.columns
                   WHERE table_schema = '{schema}' AND table_name = '{name}'
                   ORDER BY ordinal_position
               """).fetchall()

               # Get usage stats
               usage = self.sf.cursor().execute(f"""
                   SELECT COUNT(DISTINCT query_id)
                   FROM snowflake.account_usage.access_history,
                        LATERAL FLATTEN(base_objects_accessed) f
                   WHERE f.value:objectName = '{catalog}.{schema}.{name}'
                       AND query_start_time >= DATEADD(day, -30, CURRENT_TIMESTAMP())
               """).fetchone()

               datasets.append(DatasetMetadata(
                   id=f"sf.{schema}.{name}",
                   name=f"{schema}.{name}",
                   description=comment or "No description",
                   dataset_type=(DatasetType.TABLE if ttype == "BASE TABLE"
                                 else DatasetType.VIEW),
                   owner_team=self._infer_owner(schema),
                   owner_person="",
                   access_level=self._infer_access(schema),
                   schema_fields=[
                       {"name": c[0], "type": c[1], "description": c[2] or ""}
                       for c in columns
                   ],
                   tags=self._generate_tags(schema, name),
                   freshness=self._infer_freshness(schema),
                   row_count=rows or 0,
                   size_gb=round((size or 0) / 1e9, 2),
                   quality_score=0,  # Populated by quality system
                   usage_count_30d=usage[0] if usage else 0,

                       f"SELECT * FROM {schema}.{name} LIMIT 100",

                   ],
               ))

               "RAW": "Platform", "STAGING": "Platform",
   ```

   ```python
   # marketplace/search.py
   from collections import defaultdict
   import re

   class DatasetSearch:
       def __init__(self, datasets: list[DatasetMetadata]):
           self.datasets = {d.id: d for d in datasets}
           self._build_index()

       def _build_index(self):
           """Build inverted index for full-text search."""
           self.index = defaultdict(set)
           for ds in self.datasets.values():
               # Index name
               for token in self._tokenize(ds.name):
                   self.index[token].add(ds.id)
               # Index description
               for token in self._tokenize(ds.description):
                   self.index[token].add(ds.id)
               # Index tags
               for tag in ds.tags:
                   self.index[tag.lower()].add(ds.id)
               # Index column names
               for col in ds.schema_fields:
                   for token in self._tokenize(col["name"]):
                       self.index[token].add(ds.id)

       def _tokenize(self, text: str) -> list[str]:
           return [w.lower() for w in re.split(r'[_\s.]+', text) if len(w) > 2]

       def search(self, query: str, filters: dict = None,
                  sort_by: str = "relevance") -> list[dict]:
           """Search datasets by keyword."""
           tokens = self._tokenize(query)
           if not tokens:
               return []

           # Score by number of matching tokens
           scores = defaultdict(int)
           for token in tokens:
               for ds_id in self.index.get(token, set()):
                   scores[ds_id] += 1

           results = []
           for ds_id, score in scores.items():
               ds = self.datasets[ds_id]

               # Apply filters
               if filters:
                   if ("type" in filters and
                       ds.dataset_type.value != filters["type"]):
                       continue
                   if ("owner" in filters and
                       ds.owner_team != filters["owner"]):
                       continue
                   if ("access" in filters and
                       ds.access_level.value != filters["access"]):
                       continue

               # Boost certified datasets
               if ds.certified:
                   score += 5
               # Boost popular datasets
               score += min(ds.usage_count_30d / 100, 3)

               results.append({
                   "id": ds.id,
                   "name": ds.name,
                   "description": ds.description[:200],
                   "type": ds.dataset_type.value,
                   "owner": ds.owner_team,
                   "freshness": ds.freshness,
                   "quality_score": ds.quality_score,
                   "usage_30d": ds.usage_count_30d,
                   "certified": ds.certified,
                   "relevance_score": score,
               })

           if sort_by == "relevance":
               results.sort(key=lambda r: r["relevance_score"], reverse=True)
           elif sort_by == "popularity":
               results.sort(key=lambda r: r["usage_30d"], reverse=True)
           elif sort_by == "quality":
               results.sort(key=lambda r: r["quality_score"], reverse=True)

           return results
   ```
